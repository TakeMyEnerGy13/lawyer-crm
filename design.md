# Design anchor — Lawyer CRM

- **Якорь:** Attio / Linear (light) — плотный светлый SaaS-инструмент.
- **Настроение:** собранный, рабочий, спокойный.
- **Палитра:** нейтральная серая база (белый фон, gray-50 подложки, gray-200 бордеры), один акцент — сдержанный синий (#4f46e5-семейство приглушить до slate-blue). Статусы tinted-бейджами: new — blue tint, in_progress — amber tint, closed — gray/green tint.
- **Типографика:** Inter (fallback system-ui), 13–14px базовый размер, tabular-nums для счётчиков.
- **Плотность:** плотная — компактные ряды таблицы, узкие паддинги, дашборд «инструмент», не лендинг.
- **Иконки:** без эмодзи; inline SVG при необходимости.
